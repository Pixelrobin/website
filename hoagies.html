Hoagies

Hoagies is a fictional food truck that offers the best hoagies in town.

Requirements

The final product was to be a responsive static site providing a unique identity for the Food Truck.

  • Advertise the truck
  • Allow customers to easily locate it
  • Show the menu offerings
  • Provide an email sign up form
  • Include an "About" page and a blog

Technical Stack

The website is mostly writting in Pug, which is then compiled to HTML via Gulp. The JavaScript is also transpiled and bundled via Gulp using Webpack and Babel.

These tools made editing the site easier and faster. With Pug, I didn't need to have repeatable blocks of markup. With Webpack/Babel I could split up my code, use npm modules, and work with the latest JavaScript specification.

The Website

The design is intentionally Big and Bold. The pale yellow and red colors were chosen to reflect bread and meat. The curves in the logo and layout suggest the rounded shape of a good, packed hoagie.
