The Tainted Avocados

The Tainted Avocados are a fictional indie band that's just starting out.

Requirements

The band was looking for a custom responsive WordPress theme built from scratch to give them a unique look.

  • General information about the Band
  • A Blog for increased SEO
  • Upcoming events and where to get tickets
  • A place to catalog the band's albums
  • A contact form

All this had to be easily editable by the members via WordPress' capabilities.

Workflow

  1. Sketches and ideation on paper
  2. Style Tile exploration and mockups in Adobe Xd
  3. Coded in Visual Studio Code

Technical Stack

The WordPress theme and plugins were coded from scratch in PHP, the CSS was compiled from SASS (SCSS), and the JavaScript was transpiled and bundled via Webpack and Babel. All the build processes were handled with Gulp. I used WAMPServer to host the site locally while developing and BrowserSync to auto reload the browser.

Band Events

I created a custom plugin to add a simple "Event" post type to Wordpress, allowing the band to post events with a time, location and links to tickets. The date field uses flatpickr to function.
